# Start building with Tenbyte

Welcome to the **Tenbyte Developer Hub!**
This hub is designed to help you quickly and effectively integrate **Tenbyte’s edge and media infrastructure** into your projects.

Across **Tenbyte Cloud**, **CDN**, and **Vidinfra**, this hub supports you in **setting up servers**, **managing media**, and **delivering content** globally with consistent performance.

---

## Tenbyte’s Core Services

### CDN
T**enbyte’s Global CDN** speeds up content delivery using an **intelligent edge network**. With **PoPs** placed across multiple regions, it **reduces latency** and **ensures fast load times** for users anywhere in the world.

**Key features:**
- **150+ global edge** locations
- Smart Routing with **multi-CDN** redundancy
- On the fly image optimization
- Full **SSL/TLS** and **HTTP/3** support
- **Cache rules** and **edge configuration** via API
- **High-traffic** handling capacity (up to 50 Tbps)
- Real-time purging

[Learn more →](/docs/cdn)

---

### Vidinfra
**Vidinfra** is Tenbyte’s end-to-end **video infrastructure platform** for developers.
Ingest, transcode, package, deliver, and analyze video with a unified API — built for global scale and simplicity.

**Key features:**
- **`HLS`** & **`DASH`** adaptive streaming
- Smart Routing multi-CDN playback
- **AI-powered compression** for smoother playback
- Automatic optimization and encoding
- Player customization with branding and controls
- **Built-in DRM**, **signed URLs**, and **analytics**

[Explore Vidinfra →](/docs/vidinfra)

---

### Tenbyte Cloud
**Tenbyte Cloud** brings **compute**, **storage**, and **edge delivery** together to support your media and application needs. It offers a unified environment for running workloads and serving content efficiently.

**Key features:**
- **Unlimited bandwidth**
- **Backup** and **snapshot** support
- **Built-in firewall** and **DDoS filtering**
- Dedicated Private Cloud Options
- **Affordable Pricing Model**
- Seamless integration with CDN and Vidinfra

[Deploy with Cloud →](/docs/cloud)
