<Update label="28 January 2026" description="v1.0.1" tags={["Vidinfra Player", "Improvements"]}>
  ### Vidinfra Player
  
  Critical desktop fixes and mobile optimizations for improved playback experience.

  #### Added  
  • Volume and muted state persistence across videos  
  • Double-click to toggle fullscreen mode on desktop  
  • Comprehensive keyboard shortcuts and accessibility features  
  • Compact control bar layout optimized for small screens under 300px width  
  • Dark backdrop to mobile settings panel for better visibility  

  #### Enhanced  
  • Click-to-play functionality on desktop browsers  
  • Player controls auto-hide timing for better viewing experience  
  • Context menu redesigned with About and Keyboard Shortcuts views  
  • Mobile tap controls for more reliable play/pause toggling  
  • Mobile seek bar with increased touch target area and accurate scrubbing  
  • Centered play button, loading spinner, and seek buttons within the visible video area  
  • Player controls positioning in portrait fullscreen mode on mobile  

  #### Fixed  
  • Memory leaks by properly cleaning up timers on mobile devices  
  • Settings panel cutoff issues in iframes and mobile devices  
  • iOS-specific issues with controls animation and fullscreen centering  
  • Standardized all seek intervals to 10 seconds for consistent navigation  
  • Separated mute and volume controls to function independently
</Update>
