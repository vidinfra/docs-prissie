# Create and Manage API Keys

From this page, you can generate new API keys, set expiration limits, and assign access scopes to control how each key interacts with your account.

## 1. Open the API Keys Page

- From the left sidebar, under the Organization section, click **API Keys**.
You’ll land on the API Key dashboard. 

    <Frame caption="Open the API Keys Page">
        <img src="/images/account/19.png" alt="Open the API Keys Page" />
    </Frame>

- If you haven’t created any keys yet, the page will show an empty state with a **Create API Key** button.

---

## 2. Create a New API Key
- Click the **+ Create API Key** button in the top-right corner. Fill Out API Key Details

    - **Name**- Enter a clear name to identify this token.

    - **Expiration**-  Choose how long the API key will remain valid from the dropdown:

        - 1 Month
        - 6 Months
        - 12 Months
        - Unlimited

    - **Scope Access**- Select the services this API key can access.

---

## 3. Generate the Key
- Once all fields are filled in, click **Create**.

    <Frame caption="Generate API Key">
        <img src="/images/account/20.png" alt="Generate API Key" />
    </Frame>

- Your new API key will be generated and displayed once.

    <Frame caption="Generated API Key">
        <img src="/images/account/21.png" alt="Generated API Key" />
    </Frame>

---

## 4. View or Manage Existing Keys

- All previously created API keys will appear on your API Key dashboard. From here you can:

    - View key metadata 
    - Disable or delete keys
    - Create additional keys when needed
