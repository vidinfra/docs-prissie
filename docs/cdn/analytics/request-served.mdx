# Request Served

The Request Served section shows the total number of requests successfully handled by your CDN. 

    <Frame caption="Requests Served Staistics">
        <img src="/images/cdn/library-16.png" alt="Traffic Served Staistics" />
    </Frame>

## What it tells you:

- The number of video load events from viewers.
- How often content is being accessed.

## How to read it:

- The main number shows total requests.
- The percentage reflects request changes since the last time window.
