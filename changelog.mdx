<Update label="15 February 2026" description="v1.0.0"tags={["Vidinfra Webinar"]}>
  ### Tenbyte Webinar — launched

  Introducing Tenbyte Webinar — a live streaming platform for online sessions, classes, and events. Create rooms, configure presets, go live with a stage, and manage participants in real time.

  ##### ROOMS & SESSIONS
  •	Create rooms and sessions with title, description, and optional scheduled start time
  •	Session status tracking — Created, Live, and Ended — with timestamps and duration
  •	Session detail view with analytics, recordings, and attendance log

  ##### PRESETS
  •	Create reusable presets to configure stage, chat, poll, and general permissions per session
  •	Supports Vidinfra Studio and RTMP preset types

  ##### LIVE STAGE
  •	Host and on-stage participants can go live with camera, microphone, and screen share
  •	Main spotlight view with sidebar strip for all on-stage participants
  •	Audience can watch and request to join the stage; on-stage participants can leave at any time
  •	Live viewer count, session timer, and recording indicator shown in the top bar

  ##### HOST CONTROLS
  •	Mic and camera controls for host and on-stage participants
  •	Participants panel with separate on-stage and audience lists; waiting queue for stage requests
  •	Host can mute, turn off video, or remove participants from stage, and end the session for all

  ##### CHAT, POLLS & RECORDING
  •	Live chat and real-time polls available based on preset permissions
  •	Sessions recorded automatically when enabled; recordings available after session ends
  •	Attendance log with join/leave times and duration, exportable as CSV
</Update>

<Update label="28 January 2026" description="v1.0.1" tags={["Vidinfra Player", "Improvements"]}>
  ### Vidinfra Player

  #### Added  
  • Volume and muted state persistence across videos  
  • Double-click to toggle fullscreen mode on desktop  
  • Comprehensive keyboard shortcuts and accessibility features  
  • Compact control bar layout optimized for small screens under 300px width  
  • Dark backdrop to mobile settings panel for better visibility  

  #### Enhanced  
  • Click-to-play functionality on desktop browsers  
  • Player controls auto-hide timing for better viewing experience  
  • Context menu redesigned with About and Keyboard Shortcuts views  
  • Mobile tap controls for more reliable play/pause toggling  
  • Mobile seek bar with increased touch target area and accurate scrubbing  
  • Centered play button, loading spinner, and seek buttons within the visible video area  
  • Player controls positioning in portrait fullscreen mode on mobile  

  #### Fixed  
  • Memory leaks by properly cleaning up timers on mobile devices  
  • Settings panel cutoff issues in iframes and mobile devices  
  • iOS-specific issues with controls animation and fullscreen centering  
  • Standardized all seek intervals to 10 seconds for consistent navigation  
  • Separated mute and volume controls to function independently
</Update>
